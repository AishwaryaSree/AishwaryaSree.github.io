creating a site using hugo and github pages

Here is how I created a static site using hugo thems and github pages.

  • First i created two github repositories one for storing actual data and other for hosting the site.
  • Cloned a hugo theme to the storing repository.
  • You can select a hugo theme from here(https://themes.gohugo.io/)
  • Created some content for the page.
  • Added the hosting repository as a submodule.
  • Generated static files and stored it in public folder.
  • Commited the changes to the repository.
