Asciinema

Asciinema is terminal recorder which helps us to share our commands or errors easier. We can even copy paste the commands from the video which makes it much esier to run same commands in different computers.

We need to create an account in asciinema.org and install it in our computer and we are good to record. We can type the command “asciinema rec” in order to start recording and type the command “exit” to stop recording. A link will be displayed where the video is sisplayed. there are various options to share the recorded video

  1. I installed asciinema from https://asciinema.org/docs/installation
  2. I did not get any errors
