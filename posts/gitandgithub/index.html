Git and github

Git is a software that is used to track changes. Github can be used to store our code and more like webpages and can be used to collaborate.

Getting started with github Install git or gitbash-for windows. Create an account in github. We can create a repository for storing out code which is also used for collaborating if needed. We can update the code by “Push” and comminting changes. If necessary we can pull the code for another repository to your computer.
